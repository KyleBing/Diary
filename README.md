# 标题日记 vue 版

> “ 用一句话记录你最珍贵的时刻 ”

标题日记的设计初忠：能通过日记浏览过去的时间，让时间一目了然。



## 测试账号

线上访问地址： [https://kylebing.cn/diary](https://kylebing.cn/diary)


## 使用插件
- Vue
- VueRouter
- Axios
- qs
- vue-cookie
- jquery
- vue2-datepicker

项目中的样式为独立模块，未跟 `vue` 文件混在一起。
